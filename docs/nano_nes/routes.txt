# NanoCamelid NES route table — single source of truth for the always-answer
# layer. Both the Rust reference and the 6502 (via the compiled route table in
# the decode bank) resolve seeds from these rows, in this order:
#
#   A. (implicit) last two known typed words hit a non-empty trigram record
#   B. qtype + topic rows (below, no wildcard)
#   C. topic-only rows   (qtype = *)
#   D. qtype-only rows   (topic = *)
#   E. (implicit) global unknown -> bigram("?") fallback family
#   F. (implicit) emergency reseed if the answer is shorter than 6 words
#
# Syntax: QTYPE TOPIC -> SEED_W1 SEED_W2     (* = wildcard)
# Every word must exist in the corpus vocabulary; every seed pair must
# generate >= 6 words. The builder enforces both.
